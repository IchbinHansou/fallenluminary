### 1️⃣ 图片层
```css

object-cover  // 保持比例填充
```

### 2️⃣ 渐变光效层
```css
// 左上到右下渐变
bg-gradient-to-br from-white/5 via-transparent to-transparent

// 右下到左上渐变  
bg-gradient-to-tl from-transparent via-transparent to-white/3
```

### 3️⃣ 边缘发光层
```css
// 水平柔和发光
-inset-2 bg-gradient-to-r from-transparent via-white/10 to-transparent blur-sm opacity-60

// 垂直柔和发光
-inset-4 bg-gradient-to-b from-transparent via-white/5 to-transparent blur-md opacity-40
```

### 4️⃣ 边框光效层
```css
// 内层边框
border border-white/20 shadow-[0_0_30px_rgba(255,255,255,0.1)]

// 外层边框
-inset-1 border border-white/10 shadow-[0_0_50px_rgba(255,255,255,0.05)]
```
