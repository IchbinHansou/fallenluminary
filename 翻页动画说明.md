## 🎨 翻页动画效果

### 视觉特点
```
rotateY 3D旋转
↓
perspective 透视效果
↓
opacity 淡入淡出
↓
0.8秒平滑过渡
```

### 动画原理
- **进入**: 从90度旋转进入
- **中间**: 0度完全展开
- **退出**: 旋转90度离开
- **透明度**: 配合旋转淡入淡出

## 🎮 交互功能

### 自动翻页
- ✅ 第1页 → 手动点击按钮
- ✅ 第2页 → 诗歌读完后3秒自动翻页
- ✅ 第3页 → 停留供阅读和交互

### 手动翻页
- ⬅️ "上一页" 按钮（当前页 > 0）
- ➡️ "下一页" 按钮（诗歌读完后显示）
- 🔄 可以回到任意页重新阅读

### 状态重置
- 📍 滚动离开视图时重置到第1页
- 🔄 再次滚动回来可重新体验

## 🔧 技术实现

### Framer Motion
```tsx
variants={pageVariants}
initial="enter"
animate="center"
exit="exit"
```

### 3D变换
```css
perspective: 2000px
rotateY: 90deg → 0deg → -90deg
```

### 动画模式
```tsx
<AnimatePresence mode="wait">
  {/* 确保页面切换时等待上一页完全退出 */}
</AnimatePresence>
```

## 💡 与第一首诗的区别

- 自动播放所有动画

## 🌟 特色亮点

1. **沉浸式阅读**
   - 模拟真实书本翻页
   - 专注于当前页内容
   - 减少视觉干扰

2. **节奏控制**
   - 第一次翻页由用户控制
   - 之后自动引导
   - 可随时返回重读

3. **视觉优雅**
   - 3D翻页效果
   - 流畅的过渡动画
   - 统一的蓝黑月光风格

4. **信息层次**
   - 标题 → 内容 → 反思
   - 渐进式信息呈现
   - 避免信息过载
