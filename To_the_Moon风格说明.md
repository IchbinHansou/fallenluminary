# 🌙 To the Moon 风格配色方案

## 🎨 核心配色

### 背景色系
- **深空黑**: `#0a0e27` - 主背景顶部
- **午夜蓝**: `#0f1419` - 中间过渡
- **纯黑**: `#020617` - 底部深邃

### 强调色（青蓝月光系）
- **青色 (Cyan)**: `#22d3ee` → `cyan-400`
- **天蓝 (Sky)**: `#38bdf8` → `sky-400`
- **深蓝 (Blue)**: `#3b82f6` → `blue-500`
- **冰蓝 (Cyan Light)**: `#67e8f9` → `cyan-300`

### 文字色系
- **月光白**: `#f1f5f9` → `slate-100`
- **淡青色**: `#a5f3fc` → `cyan-200`
- **浅蓝色**: `#bfdbfe` → `blue-200`

---

## 🌟 视觉特点

### 1. To the Moon 游戏风格特征
- ✅ 深邃的蓝黑背景（夜空感）
- ✅ 青色/蓝色月光高光
- ✅ 忧郁、怀旧的氛围
- ✅ 星空与月光的结合
- ✅ 简洁优雅的设计

### 2. 光晕效果
- 青色月光光晕（主要）
- 蓝色星光点缀
- 天蓝色雾气扩散
- 20-30%透明度的柔和扩散

### 3. 星星配色
三种青蓝渐变：
1. 青色 → 蓝色
2. 天青 → 海蓝
3. 冰蓝 → 深蓝

### 蓝黑风格（现在）

## 📱 实际效果

### 背景
```
深黑蓝渐变 + 青色月光光晕 + 蓝色星星
→ To the Moon 式的夜空感
```

### 文字
```
青蓝渐变标题 + 月光白诗句 + 悬停变青
→ 忧郁优雅的阅读氛围
```

### 交互
```
渐显动画 + 月光跟随 + 卡片浮起
→ 沉静而流畅的体验
```

## 🌈 色调情绪

- **深蓝黑**：孤独、深邃、宇宙感

- **蓝色**：记忆、怀旧、平静的忧郁

完美契合"坠落星辰"和"错位天才"的主题！
