## 🎭 动画效果

### 月球动画
```javascript
// 月球环系统
for (let i = 0; i < 8; i++) {
  moon.rings.push({
    radius: moon.radius + (i + 1) * 20,
    opacity: 0.3 - i * 0.03,
    speed: 0.5 + i * 0.1,
    phase: Math.random() * Math.PI * 2
  })
}

// 月球表面细节
for (let i = 0; i < 50; i++) {
  moonDetails.push({
    x: (Math.random() - 0.5) * moon.radius * 1.5,
    y: (Math.random() - 0.5) * moon.radius * 1.5,
    radius: Math.random() * 8 + 2,
    opacity: Math.random() * 0.4 + 0.1
  })
}
```
