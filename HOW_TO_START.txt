🤖 如果想使用 Luna AI（可选）：

1. 访问 https://platform.openai.com/api-keys 获取API密钥

2. 在项目根目录创建文件: .env.local

3. 在文件中写入:
   OPENAI_API_KEY=sk-你的密钥

4. 重启开发服务器（Ctrl+C 然后再次 npm run dev）
