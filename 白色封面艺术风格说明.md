## 🌟 设计特点

### 1. 极简标题
- **字体**: Serif 纤细字体 (font-light)
- **颜色**: 纯黑 (slate-900)
- **大小**: 超大字号 (9xl)
- **间距**: 紧凑字距 (tracking-tight)
- **动画**: 字距展开动画

### 2. 优雅留白
- **大量空白**: 营造呼吸感
- **简洁布局**: 内容居中对齐
- **细线装饰**: 极细的分隔线
- **克制元素**: 减少不必要的装饰

### 3. 精致动画
- **缓慢淡入**: 2.5秒的图片淡入
- **字距动画**: 标题字距从宽到窄
- **浮动元素**: 20个半透明圆点
- **滚动提示**: 优雅的滚动指示器

### 4. 浮动装饰
- **数量**: 20个浮动圆点
- **颜色**: 黑色、蓝色、青色的半透明
- **大小**: 2-8px随机
- **动画**: 缓慢上下浮动

## 🔧 技术细节

### 图像处理
```tsx
// 85%透明度，保留细节
className="object-cover opacity-85"

// 白色半透明遮罩
bg-gradient-to-b from-white/60 via-white/40 to-white/70
```

### 标题动画
```tsx
// 字距动画
initial={{ letterSpacing: '0.2em', opacity: 0 }}
animate={{ 
  letterSpacing: imageLoaded ? '0.05em' : '0.2em',
  opacity: imageLoaded ? 1 : 0 
}}
```

### 浮动元素配色
```tsx
// 四种半透明颜色
rgba(0, 0, 0, 0.08)      // 黑色
rgba(59, 130, 246, 0.12) // 蓝色
rgba(34, 211, 238, 0.12) // 青色
rgba(148, 163, 184, 0.08) // 灰色
```
